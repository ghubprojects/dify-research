## Gaps

Quy ước trạng thái:

- `Open — core`: chặn core guide đạt `Review-ready`.
- `Open — deployment-specific`: không chặn core guide; chặn production validation của target liên quan.
- `Mitigated — runtime open`: guidance/control trong tài liệu đã hoàn tất, nhưng runtime rehearsal vẫn bắt buộc.
- `Deferred — deployment-specific`: chỉ kích hoạt khi target/toolchain tương ứng được chọn.
- `N/A — core out of scope`: giữ để audit phạm vi, không phải release blocker.
